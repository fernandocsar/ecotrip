
# Calculadora EcoTrip

> Estime o impacto ambiental das suas viagens de forma simples e rápida.

## Sobre o projeto

A **Calculadora EcoTrip** é uma aplicação web para estimar as emissões de CO₂ em viagens, considerando diferentes meios de transporte (carro, ônibus, avião, etc.). O objetivo é promover a conscientização ambiental de forma prática e acessível.

Este projeto foi desenvolvido durante um bootcamp na DIO, com foco em aprendizado e integração do GitHub Copilot.

## Funcionalidades
- Cálculo de emissões de CO₂ por tipo de transporte
- Interface leve e responsiva
- Uso direto no navegador, sem necessidade de instalação
- Fácil integração em outros projetos

## Tecnologias utilizadas
- JavaScript
- HTML
- CSS

## Como usar
1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Preencha os dados da viagem e veja o resultado instantaneamente.

## Exemplo de uso
Preencha a distância, escolha o tipo de transporte e confira a estimativa de emissões!
